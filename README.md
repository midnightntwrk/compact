# Compact

Compact is the Midnight Network's smart contract language.
This repository is only used to host Compact releases.

## Contributing to Compact

The project welcomes contributions.
If you would like to report a Compact bug, make a feature request, get the source code, etc.
then you should do so at the Minokawa project's [Compact repository](https://github.com/LFDT-Minokawa/compact).
